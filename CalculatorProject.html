Calculator Project

This was the last project completed for a core computer science class last fall, and it entailed all the concepts learned throughout the semester. The 'Reverse Polish Notation' calculator also follows some of the design principles that is allowed by Java, and all of the buttons are programmed to function with a keyboard and by physically clicking on them. To see the constraints and resources available for this project, click HERE

Concepts Used

This project was coded in Java using Eclipse as an IDE, which is what was used all semester. Visually, students followed Model-View-Control (MVC) principles to ensure that the program would run efficiently. For those unfamilair with MVC,it means that the model method stores data, the view method shows data, and the control method changes the visual aspect of the program based on how the data is changing.

Main Takeaways

This project was heavy on technical skills, so learning Graphical User Interfaces (GUI) practices was huge. GUI concepts introduced me to what some basic visual programs looked like, and sparked interest in other visual areas of computing such as web development. I also learned better communication skills because when I was stuck on problems, I would go to online resources or my roomate to try and find solutions.
