Personal Website

Over winter break, I gained interest in software applications that users can experience after completing the calculator project the previous semester. In order to address my curiousity, I decided to try and create a personal website that could be shown to recruiters, while also displaying more informal information about my self. After hours of learning HTML and CSS on the go and 500+ lines of code later, I came up with a minimalistic design for the site, while also displaying a diverse variety of HTML concepts to showcase my knowledge of web development.

2 Main Takeaways

HTML

HTML is the coding language used to describe the structure and organization of web pages. This is seen on my website with:

- Rows and columns of information

- Hyperlinks and image implementations

- Creation of a navagation bar

CSS

CSS is the coding language that customizes the structure and organization of web pages. This is seen on my website with:

- Different color fonts and backgrounds

- Margin size of web pages

- Vertical and horizontal alignment of text

Design Concepts

The website started out as a blank canvas, so utilizing design concepts was important to showcase the coding skills I learned, while also making the site easy for the user to read. Some design compnents included:

- Less clutter of images in order to follow a minimalistic template, allowing users to focus on text more

- Choosing lighter font and a neutral background for a contrasting interface

- Placing a navagation bar in strong power zones for easy accesibility
